### 本地启动

1. 安装依赖
```base
yarn
```
2. 启动
```base
yarn run dev
```
3. 同步 api（后端 api 更新时执行）
> 注意项目根目录下的 `openapi.config.ts` 配置
```base
yarn run genapi
```
